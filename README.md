# Workforce Forecasting & Analytics Dashboard

## Objective
To analyze workforce demand and availability using historical data and provide
actionable insights for business planning and decision-making.

## Tools Used
- Power BI
- SQL
- Excel

## Key Work
- Cleaned and transformed historical workforce data
- Built KPIs for utilization, demand vs supply, and trends
- Created interactive dashboards for leadership reporting

## Outcome
Identified upcoming talent gaps and utilization risks to support proactive planning.
